Banner Grabber

Python Banner Grabber Socket

What it does

  • Asks for target IP address and port number
  • Connects to that IP and port
  • Receives up to 1024 bytes of data (the banner)
  • Prints the banner to the screen
  • Closes the connection

Code

import socket

target = input('Enter Ipaddress: ')
port = int(input('Enter port to connect on: '))

s = socket.socket(socket.AF_INET, socket.SOCK_STREAM)
s.connect((target, port))

response = s.recv(1024)
print(response.decode())
s.close()
